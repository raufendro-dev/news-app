# News App

News App is a news aggregation app. The app is built with Flutter and data is fetched from NewsAPI https://newsapi.org/.


## Features 

- Breaking News (Home Screen) ✅
- Category News (Health, Sport, Business, Politics) ✅
- Search News ✅

## Dependencies

- bloc
- flutter_bloc 
- go_router
- http
- timeago
- webview_flutter
